# Projects Index

> Status register for all in-flight + promoted + paused + deferred projects across iCustomer.
> Update this index whenever a project's status changes. Maintained by `pm-create-spine-project` and manually as projects progress.

Status definitions:

- **Active** — work in flight; weekly progress expected
- **Paused** — work stopped temporarily; explicit reason in `state.md`
- **Complete** — work finished; files still live at `projects/<stream>/<owner>/<slug>/`
- **Promoted** — artifacts moved into permanent home (`products/<tier>/features/`, `design-guide/`, etc.)
- **Deferred** — parked indefinitely; not started or stopped without resume plan
