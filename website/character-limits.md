# Character limits

> Per-section copy constraints. The character-limit rules Claude flags when editing page shells.

## How to use

When you're editing a page shell, look up the relevant section here. If your text exceeds the limit, the design will break — either shorten the copy or flag the constraint to update.

## Pricing page (`page-templates/pricing/`)

| Section | Max chars |
|---|---|
| Hero headline | 50 |
| Hero subhead | 120 |
| Plan name | 24 |
| Plan tagline | 80 |
| Feature bullet | 70 |
| CTA button | 24 |

## Comparison pages (`page-templates/comparison/`)

| Section | Max chars |
|---|---|
| Hero headline | 60 |
| Hero subhead | 140 |
| Competitor row label | 30 |
| Feature description (vs cell) | 90 |
| Footer CTA | 32 |

## Use case pages (`page-templates/use-case/`)

| Section | Max chars |
|---|---|
| Hero headline | 70 |
| Hero subhead | 160 |
| Box title | 36 |
| Box description | 200 |
| CTA | 28 |

## Notes

- Limits are PIXEL-DERIVED from the design. They are not arbitrary.
- If a section consistently runs over, that's a signal the design needs to change — not the copy.
- For brand voice / copy conventions, see [design-guide/voice-and-copy.md](../design-guide/voice-and-copy.md) (when it lands).
