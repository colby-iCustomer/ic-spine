## Personas

- **Developer** — integrates OneSource into their own product
- **Data engineer** — pipes OneSource into their warehouse
- **AI agent builder** — calls OneSource via MCP

## Key use cases

- Resolve any identifier → immutable ID
- Enrich a contact / account via the vendor waterfall
- Use as MCP tool surface for AI agents
- Cooperative enrichment (shared signal across customers)

## What OneSource is not

- Not a CDP
- Not a campaign tool
- Not a complete decisioning platform (that's DOS)
