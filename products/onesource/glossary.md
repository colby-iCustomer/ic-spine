# OneSource — Glossary

> OneSource-specific terms.

| Term | Definition |
|---|---|
| **Immutable ID** | HMAC-SHA-256 deterministic identifier |
| **DID** | W3C decentralized identifier |
| **Match Endpoint** | The identity-resolution API. Pre-enrichment, cost-as-token model. |
| **Enrichment Waterfall** | Vendor fallback chain — provider-by-provider order |
| **Vendor Cooperative** | Shared enrichment signal across customers |
| **MCP Gateway** | Model Context Protocol surface for agent integration |
