# Decision OS — Glossary

> DOS-specific terms. For platform-structure terms, see [architecture/reference.md](../../architecture/reference.md). For brand-level terms, see [products/icustomer/glossary.md](../icustomer/glossary.md).

| Term | Definition |
|---|---|
| **Decision Engine** | Core decisioning logic. Lives in the Decision Fabric layer. |
| **Decisioning Waterfall** | L0-L6 waterfall of decisions. Third axis of the three-axis model. |
| **Scoping / Authority** | First axis — what decisions a worker is authorized to make. |
| **Reasoning Ladder** | Second axis — how much autonomy the worker has at a given level. |
| **OODA loop** | Observe / Orient / Decide / Act — the iWorker's reasoning loop. |
| **Causal AI** | The reasoning style — explanatory, not just predictive. |
