# iCustomer — Feature mind map

> High-level capability map across all three product tiers.
> Slot for auto-generated content later. For now: a manual outline.

## Capabilities (cross-product)

- **Identity** — Immutable ID, DID resolution (OneSource)
- **Enrichment** — vendor waterfall + cooperative (OneSource)
- **Decisioning** — Decision Engine, Harness, three-axis model (DOS, partially in ALoop)
- **Intelligence** — iWorkers, OODA loop, causal AI (DOS, surfacing in ALoop)
- **Activations** — push to channels via iConnect / MCP (all three)
- **Governance** — policy, permissions, Decision Traces (DOS, surfacing in ALoop)
- **Audiences** — Master + Unified Pool, suppression, cohorts (ALoop, surfacing in DOS)
- **Campaigns** — multi-channel, budgeted, time-bound (ALoop)
- **Targets + Goals + KPIs** — Goal → Target → Campaign → Playbook hierarchy (ALoop)
- **Pulse** — task-oriented recurring activities (ALoop)
